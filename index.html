About Sports-Hub

A personal hub for NFL 🦅, NBA, MLB and Soccer — live scores, standings, and a prediction game, all in your browser.

Live data comes from ESPN’s free public sports feed, fetched directly in your browser. No login, no API key, nothing to configure.

Make it yours

Soccer is set to the FIFA World Cup. Edit the LEAGUES block near the top of app.js to change favorite teams or the competition:

soccer: { ..., espnPath: 'soccer/fifa.world' }  // FIFA World Cup
// eng.1 = Premier League · usa.1 = MLS · uefa.champions = UCL
